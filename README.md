## 使用说明

1. 下载并解压到任意文件夹
2. 运行 **LdtEditor.exe**
3. **首次运行**：若 Windows 提示「未知发布者」，选择「更多信息」→「仍要运行」

### 打开 .ldt 文件

- **文件 → 打开** 或 **Ctrl+O**
- 将 `.ldt` 文件**拖入**主窗口
- 命令行：`LdtEditor.exe "D:\路径\表.ldt"`

## 功能一览

| 类别 | 说明 |
|------|------|
| 表编辑 | 打开、保存、撤销/重做、复制/粘贴 |
| 查找与筛选 | 按关键词筛选，支持正则表达式 |
| 批量替换 | 当前列查找与替换，支持预览 |
| 批量填数 | 按递增或等比规则写入数值 |
| 物品预览 | **Shift+P** 显示物品说明与头图 |
| 选图窗 | **Shift+A** 从 SPF 资源浏览图标 |
| 修改记录 | **Alt+S** 高亮已改单元格 |

---

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+O | 打开文件 |
| Ctrl+S | 保存 |
| Ctrl+Z / Y | 撤销 / 重做 |
| Ctrl+F | 功能菜单 |
| Shift+P | 物品说明预览 |
| Shift+A | 选图窗 |
| Alt+S | 修改记录高亮 |
| Alt+Z | 还原原值 |
