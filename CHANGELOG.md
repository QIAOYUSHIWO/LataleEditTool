# Changelog

本文件与程序版本（`LdtEditor/LdtEditor.csproj` 中的 `<Version>` / 「帮助 → 关于」）对齐；发版时递增版本并追加一节。

## 1.0.1 — 2026-05-27

### 选图窗（Shift+A）

- 非模态选图窗：主表换行自动刷新；双击图集格位写回 `_Icon` / `_IconIndex` 且不关窗；Esc 或窗内 Shift+A 关闭。
- SPF **图集序号**下拉：按资源包内实际存在的离散后缀切换（编号可不连续）；悬停滚轮可前后切换，切换时保留用户所选后缀。
- 索引与加载：扫描内嵌 `*_ICON##.PNG`、`ITEMETC_ICON CHINA_##` 等；标准 `*_ICON##` 禁止链序下标盲加载，减少错图。
- **行选择多选**（≥2 行）：双击图集格位将同一 `_Icon` / `_IconIndex` 批量写回全部选中行（图集以当前格行为准；块选择仍单行）；状态栏显示「批量 N 行」；撤销为整批一步。

### 修改记录（Alt+S / Alt+Z）

- 视图「修改记录」或 **Alt+S**：已改单元格高亮；悬停显示原值；**Alt+Z** 还原悬停格。
- 保存（Ctrl+S）不清空记录；关闭程序时可写入 sidecar（`%LocalAppData%\LdtEditor\change-records\`）。
- 选项可开启「自动保存修改记录」（表已落盘且无未保存改动时写入）。

### 功能菜单与表格（Ctrl+F）

- **查找 / 替换 / 插入 / 选项** 四页；窗口可拖移并限制在主窗屏幕范围内。
- **替换**：仅当前列字面量；可要求须先「预览…」后才允许「直接」批量替换。
- **插入**：按起始值与步长（加/乘）批量填数；支持前缀/后缀、原值左右拼接；可仅作用于筛选可见行、写入前预览、布尔列写入规则。
- **内嵌查找条**（视图菜单）：Enter / F3 在可见行找下一处；Ctrl+Enter 将关键词应用为行筛选（与查找页大小写/正则选项一致）。
- 表格 VirtualMode + 行筛选；列头点击对可见行排序；右键：清空格、插入空行、整行下填（Shift+V）、删行等。

### 物品说明预览（Shift+P）

- 根据当前行显示说明与头图（依赖选项中的 SPF 路径）。
- 双击 **Status** 数字可内联编辑并写回（千分制 Round×10）；多选时禁止编辑。
- 双击 **Name** / **Desc** 写回对应列；描述支持 **Ctrl+Enter** 确认。

### 解码与多开

- 打开表时字符串解码：GBK / CP949 / UTF-8 及 **UTF-8 错位链**自动择优（选项页「解码」；已移除诊断 UI）。
- 多开多个程序时，非前台实例释放全局热键（Shift+P、Shift+A），回到前台再注册，减少实例间冲突；状态栏显示热键是否注册成功（✓/✗）。

### 文档与帮助

- 重写根目录 **`README.md`**：仅面向终端用户（功能、用法、快捷键、免责）；不含构建说明与已知问题列表。
- **帮助 → 使用说明**：WebView2 + Markdown 渲染（表格、分隔线、引用等），排版接近编辑器 Preview；README 缺失时使用精简备用摘要。
- 功能菜单「插入」页提示改为用户向表述（参数自动保存）。

### 构建

- 自 **1.0.1** 起版本号与「帮助 → 关于」一致；分发见 `RELEASE.md`、`publish-release.ps1`（默认 self-contained win-x64 单文件）。
- 发布输出不再附带 WebView2 的 `*.xml` 文档文件（仅保留 `LdtEditor.exe`、`README.md`、`LICENSE`）。

## 1.0.0 — 2026-05-09

- Typed LDT 保存：按磁盘头信息测量行数据区边界，避免缓存偏移与 `_data` 不一致导致保存失败；保存时同步文件头行数（插入/删除行后一致）。
- 保存安全：可选「覆盖前 `.bak`」备份（默认开启，见功能菜单 → 选项）；保存使用临时文件再替换，降低半写入风险。
- 构建与发布：见根目录 `RELEASE.md`（framework-dependent / self-contained、`publish-release.ps1`；默认仅打 self-contained，`-Fdd` / `-FrameworkDependent` 仅打 FDD）。
- 用户文档：根目录 `README.md`（仅终端用户：配置目录、SPF、快捷键、替换流程、已知问题、免责、**著作权署名**；不含开发/发版进度）；复制到程序输出目录；**帮助 → 使用说明** 可打开阅读；维护者说明见 `RELEASE.md`、`L2_VERIFICATION.md`。
