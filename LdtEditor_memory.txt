[摘要 — 2026-05-27]
  · WinForms 编辑 LaTale LDT；主逻辑 Program.cs（MainForm、FilterToolForm）、ItemPreviewFloatForm。
  · 数据目录 %LocalAppData%\LdtEditor\：editor-settings.json、filter-tool-state.json、recent-files.json（UTF-8 无 BOM，原子写）。
  · Ctrl+F 功能菜单（查找/替换/插入/选项）；内嵌查找条 Enter 下一处、Ctrl+Enter 应用行筛选。
  · 繁体转简体：编辑 → 繁体转简体… 或 Ctrl+F 替换页「繁体→简体…」；全部文本列 + 筛选可见行，预览（仅原值/新值）后写入；LCMapString 转换。
  · Shift+A 非模态选图窗；Shift+P 物品预览；与 _Icon / _IconIndex 联动。
  · 行选择多选：双击选图窗格位批量写回相同 _Icon/_IconIndex（图集以当前格行为准；块选择仍单行；Undo 一批）。
  · 多开：MainForm IMessageFilter + WM_ACTIVATEAPP 管理全局热键注册，避免跨实例误触。
  · 选图/SPF：SpfPngArchive、SpfIconIndexCache、IconAtlasBitmapCache、IconAtlasDecodeHelper；选项「选图性能」写 editor-settings.json。
  · 批量插入：预览、仅可见行、布尔列规则；状态在 filter-tool-state.json。
  · 解码：GBK/CP949/UTF-8 三候选 + UTF-8 错位链（选项页「解码」；无诊断 UI）。
  · B1 修改记录：视图「修改记录」/ Alt+S 高亮已改格；悬停原值；Alt+Z 单格还原；Ctrl+S 不清记录；关闭可写 sidecar；选项「项目」自动保存。
  · 建议路线 ①–⑦、A1–A3 已完成；A4（搜索/最近/收藏）已撤回；选图改为 SPF 离散图集序号（见 [选图]，本地已验收）。

[配置]
  %LocalAppData%\LdtEditor\
  editor-settings.json — 解码、存盘编码、SPF、autoPersistModificationRecords、modificationTrackerVisible、公告 URL 等（选图性能项仅 JSON 默认，选项页已隐藏）
  change-records\*.json — 按 LDT 路径哈希的修改记录 sidecar（identity: 路径/大小/mtime）
  filter-tool-state.json — 功能菜单位置、查找/替换/插入状态、批量插入参数
  recent-files.json — 最近与 Pinned 路径

[主程序 Program.cs / FilterToolForm]
  Ctrl+F 查找 | 替换 | 插入 | 选项；底栏「执行查找」「清除条件」仅查找页显示。
  替换页「繁体→简体…」：LdtChineseScriptHelper（kernel32 LCMapString + LCMAP_SIMPLIFIED_CHINESE）转换全部文本列可见行，ReplacePreviewForm 仅原值/新值预览写回。
  编辑 → 繁体转简体… 同上。
  Shift+P 预览；Shift+A 选图；选图窗打开时状态栏可显示「批量 N 行」。新开 LDT 后清空筛选词再 SyncFilterColumns。
  表格 VirtualMode + 可见行筛选；保存 BuildPatchedTypedFile，可选 .bak；修改记录 sidecar 见 %LocalAppData%\LdtEditor\change-records\。
  插入：批量预览、仅筛选可见行、布尔列规则可配。替换：须先预览（可锁「直接」）。
  功能菜单拖移 ClampFilterToolFormToMainForm，锁在主窗屏幕范围内。

[选图窗 IconAtlasPickerForm]（2026-05-25 本地验收通过；2026-05-25 多选批量写回）
  非模态；主表换行自动刷新；双击格 _onApply 写回不关窗；Esc / 窗内 Shift+A 关闭。
  行选择多选（≥2 行）：双击图集格将同一 _Icon/_IconIndex 写回全部选中行；图集解析以主表 CurrentCell 行为准；状态栏提示「批量 N 行」。块选择模式仍只写绑定行。Undo 整批一步。
  顶栏：「图集序号：」+ ComboBox（仅 SPF 名索引或 DATA\GLOBALRES 中存在的后缀；编号可不连续，如 01→05 跳过 02–04）。
  滚轮悬停下拉可前后切换序号；切换保留用户所选后缀（重载时不按 _Icon%100 重置）。
  索引：BuildLogicalNameIndex 扫描内嵌 *_ICON##.PNG、ITEMETC_ICON CHINA_## 等。
  加载：标准 *_ICON## 禁止链序下标盲加载（IsBlindChainIndexIconSheet）；ITEMCHINA/542/ITEMBATTLE_CHINA 等仍专用链扫。
  预览：ItemIconPreviewLoader 同样 SnapSuffixToAvailable。
  关键类型：LaTaleIconSheet.TryResolveSheet / SpfIconIndexCache.TryGetAvailableSheetSuffixes。
